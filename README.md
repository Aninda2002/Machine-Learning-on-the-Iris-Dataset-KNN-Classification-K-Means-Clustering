# 🌸 Machine Learning on the Iris Dataset: KNN Classification & K-Means Clustering

A comprehensive machine learning project demonstrating both **Supervised Learning** and **Unsupervised Learning** using the famous **Iris Flower Dataset**.

This repository contains two complete Jupyter notebooks:

- **K-Nearest Neighbors (KNN)** implemented **from scratch** for flower classification.
- **K-Means Clustering** for grouping Iris flowers based on feature similarity.

The project is designed to help understand fundamental machine learning concepts through practical implementation and visualization.

# 🌼 Project Overview

The Iris dataset is one of the most widely used datasets for learning machine learning algorithms.

This project explores two different approaches:

### 1️⃣ Supervised Learning

Using the **K-Nearest Neighbors (KNN)** algorithm to classify flowers into their respective species based on labeled training data.

### 2️⃣ Unsupervised Learning

Using **K-Means Clustering** to automatically group flowers into clusters without using their labels.

---

# 📊 Dataset Information

The Iris dataset consists of **150 flower samples** from three different species.

### Species

- Iris-setosa
- Iris-versicolor
- Iris-virginica

### Features

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

Total Samples: **150**

Total Features: **4**

Total Classes: **3**
